# ukrainewargif

## what it does
- scrapes the ukraine/russia war 2022 wiki page for image history of the map
- downloads the images
- makes a gif of them
- cleans up after itself

## requires
- python3.6 and above
- sudoers access
- ffmpeg

## to build:
```
./build.sh
```
## to make gif
```
./make-gif.sh
```
